# Introduction to Data Science - Labs and Exercises

This repository contains my lab exercises for the course **Introduction to Data Science **. The notebooks cover topics including:

- Python exercises and tutorials
- Machine learning for customer churn prediction
- Hotel booking cancellation prediction
- Movie genre prediction and clustering

## Folder structure

- `data/` - Contains datasets used for the exercises (not included in repo due to size)
- `exercises_YT.ipynb` -  Exercises following online tutorials
- `nn_customer_churn_prediction.ipynb` - Neural network model to predict customer churn
- `hotel_booking_cancellation_prediction.ipynb` - Model to predict hotel booking cancellations
- `movies_nn_clustering.ipynb` - Neural network and clustering for movie genres
